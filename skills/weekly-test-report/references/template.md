## 缺陷统计规则

### 计数规则
1. **剔除关闭的缺陷**（状态=已关闭）
2. **剔除非 bug**（分类=非bug/重复）
3. 有效缺陷 = 总数 - 关闭 - 非bug

### 优先级映射
Excel 字段"优先级"直接对应：最高/较高/普通/较低

### 状态映射
| Excel 状态 | 周报分类 |
|-----------|---------|
| 测试提交 / 下一版本修复 | 未解决 |
| 待回归 | 待回归 |
| 持续跟踪 | 持续跟踪 |
| 待评审 | 待评审 |
| 已关闭 | 排除 |

### 严重缺陷筛选
- 优先级 = 最高 或 较高
- 用户可手动指定额外缺陷加入

## 飞书文档操作注意事项

1. ⚠️ **绝对不要用 `update_block` 修改 Page 类型块**（会清空整个文档）
2. 表格用 `create_table` + `write_table_cells` 创建
3. 图片用 `upload_image` + `after_block_id` 插入
4. 新建文档用 `feishu_doc action=create`，默认放入用户指定文件夹
5. `after_block_id` 控制插入位置
6. 修改标题：通过 `feishu_doc action=write` 重写整个文档（而非 update_block）
