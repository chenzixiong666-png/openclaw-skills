# 如何使用 XMind Generator Skill

## 📌 概述

这是一个 OpenClaw Skill，可以从 JSON 结构快速生成 `.xmind` 思维导图文件。

## 🚀 使用方法

### 方法 1：告诉我你的想法

直接和我说你想要什么样的思维导图，比如：

> "为我创建一个关于'项目管理'的思维导图，包括计划、执行、监控、收尾四个阶段"

我会帮你生成 JSON 结构并创建 `.xmind` 文件。

### 方法 2：提供 JSON 结构

如果你已经有结构化的 JSON 数据，直接传给我：

```json
{
  "title": "我的思维导图",
  "children": [
    {"title": "主题 1"},
    {"title": "主题 2"},
    {
      "title": "主题 3",
      "children": [
        {"title": "子主题 3.1"},
        {"title": "子主题 3.2"}
      ]
    }
  ]
}
```

## 📊 数据格式详解

```
{

  "children": [
```

## 📁 生成的文件

生成的 `.xmind` 文件可以用以下工具打开和编辑：

- **XMind** — 官方工具，功能最全
- **MindNode** — Mac/iOS 专用，简洁美观
- **Freemind** — 开源免费
- **Freeplane** — Freemind 的增强版
- **其他支持 XMind 的工具**

## 💡 使用建议

1. **从顶层开始** — 先定义最高级别的分类
2. **逐层细化** — 然后添加二级、三级细节
3. **保持平衡** — 同级节点数量不要超过 7-9 个
4. **定期整理** — 定期审视和优化你的思维导图
5. **视觉增强** — 在 XMind 中添加颜色、图标、备注

## 🔧 技术细节

- 文件格式：`.xmind`（ZIP 压缩 XML）
- 编码：UTF-8
- 兼容性：支持 XMind 2023 及更新版本
- 文件大小：通常很小（几 KB 到几十 KB）

## ❓ 常见问题

**Q: 生成的文件可以编辑吗？**  
A: 完全可以！生成后用 XMind 打开，可以随意编辑、添加颜色、图标等。

**Q: 支持多少层级？**  
A: 无限层级，但建议不超过 5-6 层，否则会很难阅读。

**Q: 可以导入其他软件吗？**  
A: XMind 格式被很多工具支持。生成后也可以在 XMind 中导出为 PDF、PNG、SVG 等格式。

**Q: 如何处理超大思维导图？**  
A: 我们的脚本可以处理上千个节点。但建议在 XMind 中用"智能布局"功能进行整理。
