# OpenClaw Skills

Personal collection of OpenClaw agent skills.

## Skills

| Skill | Description |
|-------|-------------|
| `figma-test-cases` | 从 Figma 设计稿分析功能并生成测试用例 Excel |
| `ones-testplan-analysis` | ONES 测试计划执行统计与阻塞分析 |
| `ones-weekly-report` | 从 ONES 项目数据生成测试周报 |
| `switchbot-openapi` | SwitchBot Cloud API 智能家居控制 |
| `weekly-test-report` | 从 Excel 缺陷数据生成飞书测试周报 |

## Usage

Place these skill directories under your OpenClaw workspace `skills/` folder.
